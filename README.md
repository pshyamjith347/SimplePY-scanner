# SimplePy Port Scanner 🛡️

A lightweight Python script to identify open ports on a target host. Designed for quick reconnaissance during security audits.

## Features
- Scans a list of high-traffic common ports (21, 22, 80, 443, 4444, etc.).
- Includes error handling for DNS resolution and connection timeouts.
- Clean, easy-to-read console output.

## How to Run
1. Ensure you have Python 3 installed.
2. Clone this repo: `git clone https://github.com/YOUR_USERNAME/SimplePy-Scanner.git`
3. Run the script: `python3 scanner.py`
4. Enter the target IP (e.g., `127.0.0.1` or `scanme.nmap.org`).

## Disclaimer
This tool is for educational and ethical testing purposes only. Do not use it against targets you do not have explicit permission to test.
